# KeenKeeper

KeenKeeper is a small web app for keeping track of close friends and important connections. I made this project so a user can see who they should contact, how long it has been, and what type of check-in is better for that person.

This is mainly a friendship reminder / relationship tracking project. The data is coming from a local JSON file.

## Technologies Used

- Next.js
- React
- Tailwind CSS
- DaisyUI
- Recharts
- Lucide React
- React Toastify
- JSON data

## Features

- Shows friend cards on the home page
- Shows status like On Track, Almost Due, and Overdue
- Details page for every friend
- Shows notes, goal days, next due date, and preferred contact method
- Quick check-in buttons for Call, Text, and Video
- Toast message after clicking a check-in button
- Timeline page for recent check-ins
- Timeline filter by All, Text, Call, or Video
- Stats page with a pie chart for Call, Text, and Video
- Responsive navbar and footer
